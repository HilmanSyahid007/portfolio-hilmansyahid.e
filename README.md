## ⚙️ Local Setup

To run this project locally:

1. **Clone the repository**
   ```bash
   git clone https://github.com/HilmanSyahid007/portfolio-hilmansyahid.e.git
   ```

2. **Navigate to the directory**
   ```bash
   cd portfolio-hilmansyahid.e
   ```

3. **Install dependencies** (if you have Node.js installed for Tailwind)
   ```bash
   npm install
   ```

4. **Run with Live Server**
   Simply open `index.html` in your browser or use VS Code's "Live Server" extension.
